# SiteSync

**AI turns your site photos into progress reports.**

Construction foremen spend 2+ hours every day on paperwork -- progress reports, safety documentation, timeline updates, and photo logs. SiteSync eliminates that burden entirely. Walk your job site, take photos with your phone, and SiteSync's AI auto-generates professional progress reports, flags safety violations, and updates project timelines. No expensive hardware. No training. Just your phone and your daily walk-through.

---

## The Problem

Construction is the least digitized major industry on the planet. McKinsey's landmark study found that construction productivity has remained flat for decades while every other industry has doubled or tripled output. The root cause? Documentation and communication failures.

- **40% of rework** on construction projects stems from poor documentation and miscommunication
- **Foremen spend 2+ hours/day** on manual paperwork instead of managing their crews
- **$31 billion/year** is wasted in the US alone on rework caused by documentation gaps
- **Safety incidents** go unreported because documentation is tedious and time-consuming
- **Project owners** get stale weekly reports instead of real-time visibility into progress
- **Disputes and claims** cost the industry billions annually due to lack of photographic evidence

The tools that exist today were built for the office, not the field. Procore costs $50K+/year and requires a project coordinator to manage. OpenSpace needs $3,000 360-degree cameras mounted on hard hats. Fieldwire handles task management but has zero AI intelligence. None of them solve the core problem: turning what happens on a job site into actionable documentation automatically.

## The Solution

SiteSync turns the phone already in every foreman's pocket into the most powerful documentation tool in construction. The workflow is dead simple:

1. **Walk your site** like you already do every morning
2. **Take photos** with SiteSync's guided walk-through mode (GPS-tagged to your floor plans automatically)
3. **AI generates everything** -- progress reports, safety findings, timeline updates, and professional PDFs
4. **Share instantly** with owners, inspectors, and your team with one tap

No new hardware. No complicated setup. No training beyond "take photos." SiteSync works because it fits into the workflow construction professionals already follow -- it just eliminates the 2 hours of paperwork that comes after.

## Quick Start

```bash
# Clone the repository
git clone https://github.com/your-org/sitesync.git
cd sitesync

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Add your API keys: SUPABASE_URL, SUPABASE_KEY, OPENAI_API_KEY, MAPBOX_TOKEN, STRIPE_KEY

# Start Expo development server
npx expo start

# Run on iOS simulator
npx expo run:ios

# Run on Android emulator
npx expo run:android
```
