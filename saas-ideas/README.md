# 30 SaaS Startup Ideas — YC-Aligned (2026-2027)

> 30 venture-backable SaaS ideas for solo technical founders, aligned with Y Combinator's documented investment priorities. Each idea is a **project folder** containing 8 comprehensive documentation files: README, tech stack, features, screens, skills, theme/design, API guide, and revenue model.

---

## How This Repo Is Organized

```
saas-ideas/
├── mobile-first/          # 10 mobile-native app ideas
│   ├── b2c/               # 5 consumer ideas
│   │   └── 01-fieldlens/  # Each project is a folder with 8 MD files
│   │       ├── README.md          # Overview, YC alignment, market opportunity
│   │       ├── tech-stack.md      # Best tech stack for profitability & scale
│   │       ├── features.md        # All features, user stories, dev timeline
│   │       ├── screens.md         # Every screen, UI elements, functionality
│   │       ├── skills.md          # Skills needed to build THIS project
│   │       ├── theme.md           # Color scheme, typography, design system
│   │       ├── api-guide.md       # APIs, pricing, alternatives, code snippets
│   │       └── revenue-model.md   # Pricing, $1M MRR math, unit economics
│   └── b2b/               # 5 business ideas
├── desktop-first/         # 10 desktop-native app ideas
│   ├── b2c/               # 5 consumer ideas
│   └── b2b/               # 5 business ideas
└── web-first/             # 10 web app ideas
    ├── b2c/               # 5 consumer ideas
    └── b2b/               # 5 business ideas
```

## Research Foundation

All 30 ideas are aligned with Y Combinator's documented 2026-2027 investment priorities:

- **YC Spring 2026 RFS**: Cursor for PMs, AI-Native Hedge Funds, AI-Native Agencies, Stablecoin Finance, AI for Government, Modern Metal Mills, AI for Physical Work
- **YC Vertical AI Thesis**: "Vertical AI agents could be 10X bigger than SaaS" — Jared Friedman, Lightcone Podcast
- **S25 Demo Day Trends**: Agentic AI (50%+ of batch), compliance automation, executive AI agents, voice AI, democratized development
- **Garry Tan**: "YC wants founders who treat AI agents not as features but as the core operating system of brand-new companies"
